## DayTradder With Treasure Direct
![icon](https://badgen.net/badge/version/1.0.0#DDDDD?)

This project have as objective managed the market marking utiliting mensage of e-mail, functing as chatbot.
The user will send the mensage of email with attch in format 'xlsx' contaning information of the buys and sales ofs titles the treasure direct.
The application will send the e-mail graphic the market marking with moving averages and indication if the moment is of buy or the sales.

## Technologies

The project was created with:
* Python version: 3.12.2
* Poetry version: ^1.8.4

## Setup

We used poetry to install this project:

```
git clone https://github.com/yurifsilveira/marking-on-the-market.git

poetry install

```
